Some of my recent projects

Route calculator

This route optimizer asks you about your living place and working place. After the required information filled, optimal route is calculated for a car and for a bike(using a map box routing machine). User gets the distance to work, spendings on fuel and emission of Co2 yearly(with the assumption that user goes to the work 252 days in a year). The user also gets an elevation chart of the bike path.

Measure angle on a Web map

This tool measures the angle between two vectors on a map projection. Click on the small button on the left upper corner would activate drawing first vector and activate calculation of angle between the first vector and second starting from а vertex when first drawn. Right-click enables calculation for the opposite angle(outer or inner).

Vehicle counter

Data for this tiny app is obtained from OpendataDK, published by Aarhus Municipality. Real-time data is delivered with the number and average speed of vehicles. In this case, it is represented as a bubbling circle where size depends on vehicle numbers and color of average speed(blue as lower than 50km/h, red more than 80km/h and white for values in between). More: https://portal.opendata.dk/organization/about/city-of-aarhus

Slider with randomly generated points

This small project randomly generates 100 points in map view extent and assigns numbers from 0 – 10 as point attributes. These numbers are used as a reference for the slider. Depending on which number point of the slider is points with that number(or less) will be shown on the map. The slider runs automatically until the user starts an interaction with the slider.

London area through the years

Map showing the area of London through 200 years period with a chart representing population growth. Fully responsive and dynamic.

Snap to the line and new quadratic geometry on Sketch Tool

As a student intern, I worked on the ‘Similix Sketch Tool’ and new advanced features of the application. It included changes in the behavior of geometries in a Web environment using ESRI API 4.6/9, Angular 5 and .Net framework. Mostly, this was based on implementing new algorithms in JavaScript and C# and conducting various geospatial analysis. Because of company privacy rules, I am unable to show code I wrote for this project.

GIS Solution - Locating the Optimal Place for Pharmacies

Three colleagues and I built a Web application from scratch for determining the best location in the Norrebro area of Copenhagen for a new pharmacy store based on multiple criteria analysis. The whole process involved data collecting, evaluation, and representation of results. We used SQL and PostGIS, Python and JavaScript in order to achieve results.
